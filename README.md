## How It Works

Every Playbook is a complete operating procedure:

- **Trigger** — client texts a phrase via WhatsApp, SMS, or Slack
- **Agents** — assigned team executes the full SOP autonomously
- **Deliverables** — structured output delivered back to the client thread
- **Quality Gates** — automated checks before anything ships
- **Escalation** — rules for when humans get looped in

No dashboards. No configuration. Just results in the thread.
